EB-BILL-GENERATOR is a Java console-based application designed to simulate an electricity bill generation system.
The primary goal of this project is to improve proficiency in core Java concepts, with strong emphasis on OOP pillars, exception handling, and collections.

Key highlights:

Applied OOP concepts for modular and maintainable code

Used Java Collections Framework for data storage and manipulation

Implemented exception handling for user input validation and runtime safety
This project serves as a hands-on learning exercise to bridge theoretical Java concepts with practical application.
